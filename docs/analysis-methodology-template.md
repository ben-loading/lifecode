## 一、八字分析核心方法

### 1. 四柱结构解读
```
年柱：祖辈、早年环境、原生家庭
月柱：父母宫、兄弟姐妹、青年期社交
日柱：核心自我、配偶宫、中年格局
时柱：子女、晚年、事业成就
```

### 2. 五行平衡分析
- **日元强弱**：判断身强/身弱
- **喜用神**：确定需要的五行（生扶或克泄）
- **忌神**：确定不利的五行

### 3. 十神关系
- **比劫**：自我意识、竞争力
- **食伤**：表达能力、创造力
- **财星**：财富观、异性缘
- **官杀**：事业心、权威感
- **印星**：学习力、贵人运

### 4. 格局判断
- **从格**（从强、从弱）
- **化格**（化气格）
- **专旺格**
- **普通格局**

## 二、紫微斗数分析核心方法

### 1. 命盘三要素
```
命宫：人生主线、核心性格
身宫：后天努力方向、中晚年重心
福德宫：内心世界、精神追求
```

### 2. 十二宫位系统
| 宫位 | 主管领域 | 重点关注 |
|-----|---------|---------|
| 命宫 | 核心自我 | 主星组合、四化 |
| 兄弟 | 手足关系、合作 | 与命宫关系 |

| 福德 | 精神享受 | 与命宫三方 |

### 3. 星曜亮度系统
```
庙 > 旺 > 得 > 利 > 平 > 不 > 陷
```
- **庙旺**：星曜力量最强，正面作用明显
- **得利平**：中等力量，平衡状态
- **不陷**：力量受损，负面影响加大

- **化禄**：财运、贵人、机遇
- **化权**：权力、掌控、执行力
- **化科**：名声、考试、文书
- **化忌**：阻碍、波折、破财

### 5. 六吉六煞
**六吉星**：文昌、文曲、左辅、右弼、天魁、天钺
**六煞星**：擎羊、陀罗、火星、铃星、地空、地劫

## 三、八字与紫微结合分析法

### 方法 1：双重验证法
```
八字看【先天格局】 → 紫微看【后天发展】
```
- 八字：判断命主的**天赋优势**和**五行喜忌**
- 紫微：判断命主的**努力方向**和**人生节奏**

**示例**：
- 八字日元为火，身强，喜金水（制约/泄秀）
- 紫微命宫武曲（金）+ 贪狼（水），正好对应喜用神
- **结论**：事业选金融（武曲）+ 投资（贪狼）可顺应天命

### 方法 2：宫位对应法
| 八字领域 | 对应紫微宫位 | 综合分析重点 |
|---------|------------|-------------|
| 日柱（自我） | 命宫 | 核心性格、人生主线 |
| 财星 | 财帛宫、田宅宫 | 财富格局、赚钱方式 |
| 官杀 | 官禄宫 | 事业成就、权力欲望 |
| 印星 | 父母宫、福德宫 | 学习能力、精神追求 |
| 食伤 | 子女宫 | 创造力、表达欲 |
| 配偶星 | 夫妻宫 | 婚姻状态、伴侣特质 |

### 方法 3：时间轴整合法
```
八字大运（10年） + 紫微大限（10年） → 人生阶段判断
八字流年天干地支 + 紫微流年四化 → 年度运势判断
```

**综合分析步骤**：
1. 八字看大运天干地支与日元关系（生扶/克泄）
2. 紫微看大限走到哪个宫位，主星状态如何
3. 两者叠加：若八字走好运 + 紫微大限吉，则该10年必有成就
4. 若八字走忌神运 + 紫微煞星聚集，则需谨慎保守

### 方法 4：五行能量映射法
```
八字五行分布 → 紫微五行局 + 主星五行属性
```

**五行对应表**：
| 五行 | 八字十神 | 紫微主星 |
|-----|---------|---------|
| 木 | 比劫、食伤（甲乙） | 天机、巨门、贪狼 |
| 火 | 食伤、财星（丙丁） | 廉贞、破军 |
| 土 | 财星、官杀（戊己） | 天府、天相 |
| 金 | 官杀、印星（庚辛） | 武曲、七杀、太阴 |
| 水 | 印星、比劫（壬癸） | 太阳、天同、天梁 |

**应用示例**：
- 八字缺水，需要水来调候
- 紫微财帛宫在亥（水宫），有太阴（水星）
- **建议**：从事流动性行业（贸易、金融）可补水

## 四、七维能量综合评估法

### 自我维度（命宫 + 日元）
1. 八字看日元强弱：
   - 身强 → 自我意识强，主观性强 → 评分 80+
   - 身弱 → 容易受他人影响 → 评分 60-
2. 紫微看命宫主星：
   - 紫微、天府、七杀独坐 → 帝王气场 → 评分 90+
   - 空宫或杂星多 → 缺乏主见 → 评分 50-
3. **综合评分**：取两者平均值

### 财富维度（财星 + 财帛宫）
1. 八字看财星：
   - 财星透干且有力 → 财运旺 → 评分 80+
   - 财星被克或入墓 → 财运差 → 评分 50-
2. 紫微看财帛宫：
   - 武曲、天府、禄存 → 财源稳定 → 评分 85+
   - 破军、天空、地劫 → 破财严重 → 评分 40-
3. **综合评分**：若八字财弱但紫微财强，则后天努力可补先天不足

## 六、关键原则

1. **八字为体，紫微为用**：八字决定先天格局（体质），紫微决定后天发展（用途）
2. **互补原则**：八字缺什么，紫微能否补上？
3. **共振原则**：八字喜用神与紫微主星五行一致时，力量倍增
4. **时间轴原则**：大运+大限同步看，流年+流年四化同步看
5. **宫位对应原则**：八字十神→紫微十二宫，形成双重验证
